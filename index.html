Features

💥

Destructible Terrain

Every shot reshapes the battlefield. Procedural Midpoint Displacement generates a unique landscape each round; explosions carve physically accurate circular craters in real time.

🎇

Multi-layer Particles

2 048-particle pool. Explosions render in three passes: additive white flash → orange shockwave ring → physics debris with gravity and friction → rising smoke plume.

🎵

Retro Synth Audio

Zero audio files. A real-time SDL3 callback synthesises every sound: a descending square-wave cannon shot and an exponentially-decaying white-noise explosion burst.

🌬

Wind Physics

Randomised wind force changes every turn and affects projectiles and smoke particles alike. Factor it in — or aim poorly and blame the weather.

🛒

Shop & Economy

Land hits to earn credits. Spend them between rounds on Nukes, Cluster Bombs, and Penetrators. Strategic shopping is the difference between victory and rubble.

📱

Touch + Keyboard

Left-half drag to aim, right-half drag to set power, centre-bottom tap to fire. The same binary runs identically on a phone and a desktop with SDL3 logical presentation.

Under the Hood

Language C++20 Smart pointers · RAII · constexpr · std::numbers
Library SDL3 Windowing · Renderer · AudioStream API
Build CMake 3.20+ FetchContent fallback · ASAN in Debug
Resolution 640 × 360 SDL_LOGICAL_PRESENTATION_INTEGER_SCALE
Timestep 60 Hz fixed Accumulator loop · 250 ms spiral guard
Audio Real-time PCM Square wave + xorshift white noise · mutex-safe queue
Terrain Midpoint Displacement Heightmap · Deform(cx,cy,r) concave crater
Particles 2 048-pool Additive + alpha passes · round-robin allocation

Build in 3 commands

$ git clone git@github.com:xavierbasc/artillery-tank-app.git
$ cmake -B build -DCMAKE_BUILD_TYPE=Release
$ cmake --build build -j$(nproc)

SDL3 is downloaded automatically via FetchContent if not found on your system.
